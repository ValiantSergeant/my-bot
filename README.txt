===== BOT DISCORD KAYO =====

SETUP RAPIDE:

1. INSTALLE LES DÉPENDANCES:
   pip install -r requirements.txt

2. LANCE LE BOT:
   python bot.py

C'est tout! Le token est dans .env

===== COMMANDES =====

+dmall [message]
→ Envoie un message privé à tous les membres

+bankai
→ Supprime tous les salons et en crée 20 nouveaux "KAYO KAYO"
